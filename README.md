# 🐧 咕咕嘎嘎 - 桌面宠物

一只可爱的企鹅桌面宠物，会呼吸、散步、说话、睡觉，陪你度过每一天！

## ✨ 功能

| 功能 | 说明 |
|------|------|
| 🎈 **呼吸动画** | 待机时上下浮动，眼睛会四处看 |
| 🖱️ **拖拽移动** | 按住拖动咕咕嘎嘎到你喜欢的位置 |
| 👆 **点击互动** | 戳戳企鹅，它会弹跳、摇晃脑袋、冒爱心~ |
| 💬 **随机话语** | 开心时、无聊时、被戳时都有不同台词 |
| 🚶 **散步模式** | 右键托盘开启，咕咕嘎嘎会自己在桌面上散步 |
| 🎯 **跟随模式** | 右键托盘开启，咕咕嘎嘎会跟着你的鼠标走 |
| 😴 **自动睡觉** | 5分钟不理它就会打盹 |
| 📌 **系统托盘** | 最小化到托盘，随时唤出 |

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动桌面宠物
npm start
```

## 🎮 操作

- **拖拽** - 按住拖动企鹅
- **单击** - 戳戳互动
- **双击** - 用力戳（特殊反应）
- **右键托盘** - 打开菜单（散步/跟随/退出）
- `Ctrl+Shift+W` - 开关散步模式
- `Ctrl+Shift+F` - 开关跟随模式

## 🎨 自定义图片

把你的企鹅角色图片放到 `assets/penguin.png`，应用会自动使用你的图片替换 CSS 绘制的默认形象。

推荐：透明背景 PNG，尺寸约 140×180px 或更大。

## 🛠 技术栈

- **Electron** - 透明无边框桌面窗口
- **HTML/CSS/JS** - 宠物渲染和动画
- **CSS Animations** - 所有动画（呼吸、弹跳、摇晃等）
- **纯 CSS 企鹅角色** - 无需图片也能看到咕咕嘎嘎

## 📁 项目结构

```
.
├── main.js          # Electron 主进程（窗口管理、托盘）
├── preload.js       # 预加载脚本（安全桥接）
├── index.html       # 宠物窗口
├── renderer.js      # 宠物行为逻辑
├── styles.css       # 样式和动画
├── package.json     # 项目配置
└── assets/
    └── penguin.png  # 你的自定义角色图（可选）
```
